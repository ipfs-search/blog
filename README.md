# blog.ipfs-search.com

Created using [Gridsome](https://gridsome.org/) and [Gridsome Casper V3 Starter](https://gridsome.org/starters/gridsome-casper-v3-starter/).

## Requirements
* [Node](https://nodejs.org/)

## Developing locally
### Install dependencies
```sh
npm i
```

### Running dev server
```sh
npm run develop
```

### Running build
```sh
npm run build
```
